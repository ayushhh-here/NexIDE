## 🧠 What is NexIDE?

**NexIDE** is a blazing-fast, AI-integrated browser-based IDE — no installs, no setup, just code.

Built with **Next.js App Router**, **WebContainers**, **Monaco Editor** (the same engine powering VS Code), and **local LLMs via Ollama** — NexIDE brings the full developer experience directly into your browser.

> 🚀 **This is my first major fullstack project** — built to deeply understand how real-world developer tools work under the hood.

---

## ✨ Features

| Feature | Description |
|---|---|
| 🔐 OAuth Login | Google & GitHub login via NextAuth |
| 🎨 Modern UI | TailwindCSS + ShadCN — Dark/Light mode |
| ⚡ WebContainers | Run Node.js apps instantly in the browser |
| 🤖 AI Assistant | Local LLM via Ollama — autocomplete & chat |
| 📝 Monaco Editor | VS Code-grade editor with syntax highlighting |
| 📁 File Explorer | Create, rename, delete files & folders |
| 🧩 8+ Templates | React, Next.js, Vue, Express, Hono, Angular & more |
| 💻 Embedded Terminal | Full xterm.js terminal in the browser |

---

## 🛠 Tech Stack

Frontend   → Next.js 15, TypeScript, TailwindCSS, ShadCN UI

Editor     → Monaco Editor (VS Code engine)

Runtime    → WebContainers API (runs Node.js in browser)

AI         → Ollama (local LLMs)

Auth       → NextAuth.js (Google + GitHub OAuth)

Database   → Prisma ORM

Terminal   → xterm.js

## 🚀 Getting Started

```bash
# Clone the repo
git clone https://github.com/ayushhh-here/ai-code-editor.git
cd ai-code-editor

# Install dependencies
npm install

# Run the dev server
npm run dev
```
